INFORMATION

This program was originally created in Jupyter Notebook. It leveraged the VOSK Offline Speech Recognition API, and was programmed in Python.

I knew porting it to this portfolio site was going to be inconvenient, but felt like an information box + a download would suffice.

Later, I wanted to give porting the finished program a go, and starting to understand the limitations of existing methods to run python programs on the front-end of websites.

I did some research, and learned about using the Web Speech API. This API solved a lot of the problems that hosting the VOSK API locally had, and was much more accurate due to it using a larger model.

Browser support for the API isn't perfect, so there may be issues in trying to run this program. Error messages should display if you run into issues.
