SQL Notebooks

While the main goal with this website is to display things I've made in the past in an in-website nicely interfaced format, some programs are either generally unportable or better viewed in the format of a Jupyter Notebook.

Provided below is a list of a few of the SQL projects that I have worked on while picking up the language, and over time I plan to add more whenever I locate the files/create new projects.

IMPORTANT: Each program has a Google Colab upload, a raw Jupyter Notebook upload, and an in-website notebook viewer. Google Colab does not natively support running any Jupyter Notebooks besides ones written in Python. If you want to run the programs instead of just viewing the code, I recommend downloading SQLite3 and copy-pasting the code into it.
